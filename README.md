# WebApp-Mascotas
Aplicación web para la adopción de animales en situación de calle en entorno universitario.
  - Gestión de usuarios
  - Gestión de roles
  - Gestión de animales en situación de calle
  - Gestión de avisos de adopción

Tecnologías de software:
  - PHP Laravel 5.8
  - Bootstrap
  - MySQL
  - HTML
  - CSS

date dev: 2019
